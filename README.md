# Carro do Vitinho
A [Godot engine](https://godotengine.org/) game based in [CarGame](https://github.com/NotHolst/CarGame).

### Description
 
Vitinho is a nice Brazillian guy who have a awesome pimped car. His friend [PC Siqueira](https://www.youtube.com/user/maspoxavida) made a campaign to make the car greater than it already was.
This game is nothing but an acknowledgement of the awesomeness of Vitinho's car. It deserves, it conquered its glory!

So help vitinho and his gang to save Brazil with his awesome car through this awesome game with your awesome abilities. We count on you!
